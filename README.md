 Projects

 🔮 AI Voice Agent
A live browser-based voice agent connected directly to the ElevenLabs API. Speak naturally into your microphone and get real-time voice responses.

Stack: ElevenLabs API · Browser Web Audio API · Microphone access

---

 🎥 UGC Ads Creator
Upload a product image and watch an AI agent generate a full animated video advertisement. Combines computer vision with AI video generation.

Stack: Computer Vision · AI Video Generation · Agentic Workflow

---

 🦜 Captain's Parrot
An AI chatbot (Polly) that answers recruiter questions about Aakanksha's skills, experience, and background. Features streaming typewriter-style responses.

Stack: AI Chat API · Streaming responses · Custom persona

---

 🧭 Navigator's Console
A fully functional SSH-style terminal built in the browser. Supports commands like neofetch, ls skills/, sudo hire, ping, and more, with arrow-key history navigation.

Stack: Vanilla JS · Custom command parser · Terminal emulation

---

 ⚓ Recruiter Mode
Same person, two presentations. A toggle switch flips between a pirate-themed CV and a polished professional resume with a 3D card flip animation.

Stack: CSS 3D transforms · Vanilla JS · Responsive layout
